Uppbygging forms

  • Form er búið til með <form> tagi
  • Inniheldur input-reiti
  • Megin eiginleikar: action og method

action og method

  • action: Hvert gögnin fara (URL eða skrá)
  • method: Hvernig gögnin eru send (GET eða POST)
  • GET: Gögn í URL
  • POST: Gögn send í bakgrunni (ekki sýnileg í URL)

Algeng input-týpur

Tegund Notkun
text Texti, t.d. nafn
password Lykilorð
email Netfang
number Tala
checkbox Kross (já/nei)
radio Valhnappur
submit Senda form

Mismunandi eiginleikar input elements

  • name – lykill fyrir gagnaflutning
  • required – skyldusvæði
  • placeholder – dæmi um hvað á að fylla út
  • min, max – takmörk á tölum o.fl.

Hvernig virkar sending?

  • Notandi fyllir út
  • Ýtir á "Senda" hnapp
  • Vafri sendir gögn til action með aðferð method
  • Gögn birtast í URL (GET) eða send í bakgrunni (POST)

Dæmi um URL með GET gögnum

thankyou.html?name=Jón&email=jon@example.com

Lesa gögnin með JavaScript

const params = new URLSearchParams(window.location.search);
const name = params.get('name');
console.log(name);
